# VS Comparison Article Template

**For articles like: "[Model A] vs [Model B]"**

## Sections (in this order)

1. **Quick Answer callout** — One-sentence verdict: "[Model A] wins for [use case X]; [Model B] wins for [use case Y]"

2. **Model Overview** — Brief intro to both models (1 paragraph each)
   - 📌 From HF model cards `--- Model ---`

3. **Specifications Comparison** (table) — Parameters, architecture, context window, release date, license
   - 📌 From HF model cards

4. **Benchmark Comparison** (table + chart) — Key benchmarks side by side (pick 6-8 most relevant)
   - 📌 From HF README `ALL BENCHMARKS`

5. **Pricing Comparison** (table) — API pricing from all available providers
   - 📌 From `>>> USE THIS PRICE <<<` and `--- OpenRouter Provider Data ---`

6. **Use Case Recommendations** — When to choose Model A vs Model B (scenario-based)

7. **Conclusion + Key Takeaways + FAQ**

## Constraints

- MUST have a clear verdict/thesis — never "both are good, it depends"
- Every comparison claim must cite specific numbers
- Minimum 2 benchmark charts/tables

## Thesis Template

"[Model A] is the better choice for [scenario X] because [data point]; choose [Model B] when [scenario Y]"
