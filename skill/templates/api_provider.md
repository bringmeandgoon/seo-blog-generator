# API Provider Article Template

**For articles like: "[Model] API providers", "[Model] API pricing comparison"**

## Sections (in this order)

1. **Quick Answer callout** — "Best value: [Provider] at $X/M tokens. Fastest: [Provider]. Most features: [Provider]."

2. **Model Overview** — Brief intro + key specs
   - 📌 From HF model card `--- Model ---`

3. **Provider Comparison** (main table) — All providers with pricing, latency, features
   - 📌 From `>>> USE THIS PRICE <<<` and `--- OpenRouter Provider Data ---`

4. **Novita AI Deep Dive** — Setup, code example, unique features
   - 📌 From `--- Novita AI Integration Guide ---`

5. **Provider-by-Provider Breakdown** — 2-3 sentences per provider with pros/cons

6. **Cost Analysis** — Monthly cost estimates for different usage tiers (table)

7. **How to Choose** — Decision framework based on use case

8. **Conclusion + FAQ + SEO Titles**

## Constraints

- Pricing MUST come from pre-search data only — never from memory
- Include at least 3 providers in comparison
- Novita AI section should be the most detailed

## Thesis Template

"For [use case], [provider] offers the best [cost/speed/reliability] trade-off at $X/M tokens"
