# Platform Article Template

**For articles like: "[Model] on Novita AI", "deploy [Model] on [Platform]"**

## Sections (in this order)

1. **Platform Overview** — what is this platform, why it's relevant for this model

2. **Getting Started** — account setup, authentication, API key generation

3. **API Usage** — code examples (Python, OpenAI SDK), streaming, function calling
   - 📌 Setup from `--- Novita AI Integration Guide ---`
   - 📌 Pricing from `>>> USE THIS PRICE <<<`

4. **Pricing** — detailed pricing breakdown, cost estimation for common use cases

5. **Performance** — latency, throughput, uptime data
   - 📌 From `--- HuggingFace Inference Providers ---` or `--- OpenRouter Provider Data ---`

6. **Limitations** — context window constraints, rate limits, known issues

7. **Alternatives** — brief mention of other platforms/providers

8. **Conclusion** — recommendation and next steps

9. **FAQ** — 5 practical questions

## Thesis Template

"[Platform] offers the most [cost-effective/fastest/easiest] way to use [Model] for [use case]"
