When you need to modify the “Docker Image”, “Container Start Parameters” or “Container Environment Variables” of a GPU instance, you can use Novita AI Upgrade Instance feature. Here’s how to use this feature.
- While upgrading the instance, the instance will be reset, but the resources originally occupied by the instance will be retained.
- When upgrading the instance, the platform will save your system disk data, but if the data is incompatible with the upgraded instance system, the data cannot be successfully restored to the new instance.
- Go to the Instances page and locate the instance you want to upgrade. Click “More Operations - Upgrade”.
- In the pop-up edit dialog, modify the image and other configurations, and click the “Upgrade” button after confirming everything is correct.
