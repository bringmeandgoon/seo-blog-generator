Novita Agent Sandbox CLI is a command line tool that allows you to list, kill running sandboxes , and manage sandbox templates .
## ​Installation
You can install the CLI globally using npm:
Bash Copy
```
npm i -g novita-sandbox-cli
```
Additionally, some new advanced features will be first released in the Beta versions of CLI tool, which you can install using the following methods. If you encounter any issues during use of these Beta features, please feel free to contact us .
## ​Install Beta CLI
Bash Copy
```
npm i -g novita-sandbox-cli@beta
```
