## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Response
​ availableBalance string The user’s credit balance, unit is 0.0001 USD.
​ cashBalance string The remaining balance of your top-up. The Unit is 0.0001 USD.
​ creditLimit string Your credit limit (i.e., the maximum amount you can owe), unit is 0.0001 USD.
