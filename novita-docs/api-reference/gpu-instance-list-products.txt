## ‚ÄãRequest Headers
‚Äã Content-Type string required Enum: application/json
‚Äã Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ‚ÄãQuery Parameters
‚Äã clusterId string Filter by specified cluster ID. String, length limit: 0-255 characters.
‚Äã gpuNum integer Filter by number of GPUs. Integer, valid range: [0, 8].
‚Äã productName string Filter by product name (fuzzy match). String, length limit: 0-255 characters.
‚Äã minCpuPerGpu integer Filter by minimum CPU cores per GPU. Integer, value must be greater than or equal to 0.
‚Äã minMemoryPerGpu integer Filter by minimum memory per GPU (GB). Integer, value must be greater than or equal to 0.
‚Äã minRootFSSize integer Filter by minimum root filesystem size per GPU (GB). Integer, value must be greater than or equal to 0.
‚Äã billingMethod string Filter by billing method. Options:
- onDemand: Pay-as-you-go instance (default)
- monthly: Subscription instance (monthly or yearly)
- spot: Spot instance billing
## ‚ÄãResponse
‚Äã data object[] required GPU product information. Hide properties ‚Äã id string required Product ID. ‚Äã name string required Product name. ‚Äã cpuPerGpu integer required Number of CPU cores per GPU. ‚Äã memoryPerGpu integer required Memory size per GPU (GB). ‚Äã diskPerGpu integer required Disk size per GPU (GB). ‚Äã availableDeploy boolean required Whether this product can be used to create an instance. Values:
- true: The product can be used to create an instance.
- false: Insufficient resources, cannot create an instance.
‚Äã minRootFS integer required Minimum available root filesystem size (GB). ‚Äã maxRootFS integer required Maximum available root filesystem size (GB). ‚Äã minLocalStorage integer required Minimum available local storage size (GB). ‚Äã maxLocalStorage integer required Maximum available local storage size (GB). ‚Äã regions [string] required Available clusters. Indicates that this product is only available in the specified clusters. If the list is empty, the product is available in all clusters. ‚Äã price integer required Price for creating a pay-as-you-go instance with this product. ‚Äã monthlyPrice object[] required Price for creating a subscription (monthly or yearly) instance with this product. Hide properties ‚Äã price integer required Unit price for the subscription instance. ‚Äã month integer required Subscription duration, in months. ‚Äã billingMethods string[] required The billing methods supported by this product. Valid values:
- onDemand: Pay-as-you-go billing
- monthly: Monthly subscription billing
- spot: Spot instance billing
‚Äã spotPrice string Spot billing instance price. ‚Äã inventoryState string Product inventory status:
- none: Out of stock
- low: Low stock
- normal: Normal stock
- high: Sufficient stock
