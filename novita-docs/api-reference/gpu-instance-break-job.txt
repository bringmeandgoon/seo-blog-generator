## ​API Description
Forcefully interrupt a running job.
## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ jobId string required The job ID to be interrupted. String, length 0–255.
## ​cURL Example
Copy
```
curl --location --request POST 'https://api.novita.ai/gpu-instance/openapi/v1/job/break' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {{API_KEY}}' \
--data-raw '{
"jobId": "10wi9visv0rqt57w"
}'
```
