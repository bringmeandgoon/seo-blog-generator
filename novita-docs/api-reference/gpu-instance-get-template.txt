## ‚ÄãRequest Headers
‚Äã Content-Type string required Enum: application/json
‚Äã Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ‚ÄãQuery Parameters
‚Äã templateId string required ID of the template being queried.
## ‚ÄãResponse
‚Äã template object Template settings. Show properties ‚Äã Id string Template ID. ‚Äã name string Template name. ‚Äã readme string Template README content (in Markdown format). ‚Äã type string Template type.Enum: instance ‚Äã channel string Template channel.Enum: private ‚Äã image string Docker image address for instance startup. ‚Äã imageAuth string Image repository authentication ID for private images. ‚Äã startCommand string Startup command for the instance. ‚Äã entrypoint string Instance startup entrypoint. ‚Äã rootfsSize integer Rootfs storage (GB). ‚Äã ports object[] Exposed ports settings. Show properties ‚Äã type string Exposed port types.Enum: http, tcp ‚Äã ports [integer] Exposed ports (maximum 10). ‚Äã envs object[] Environment variables injected into instance. Show properties ‚Äã key string Environment variable key. ‚Äã value string Environment variable value. ‚Äã tools object[] Template built-in tools (only available for official templates). Show properties ‚Äã name string Tool name. ‚Äã describe string Tool description. ‚Äã port integer Tool port. ‚Äã type string Tool protocol type. Enum: http , tcp ‚Äã createdAt string Template creation timestamp (Unix seconds). ‚Äã recommendCards object[] Recommended GPU card specs (only available for official templates). Show properties ‚Äã gpuSpecId string GPU spec ID. ‚Äã cardNum string Recommended card number. ‚Äã minCudaVersion string Minimum required CUDA version, e.g. 11.8, 12.4.
