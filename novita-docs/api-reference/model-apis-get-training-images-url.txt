This API provides an S3 pre-signed uploading URL for training images.
## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ file_extension string required Enum: png , webp , jpeg
## ​Response
​ assets_id integer The asset ID.
​ upload_url string The S3 pre-signed uploading URL.
​ method string The method for uploading.
Enum: PUT
​ headers object Show properties ​ Host object Show properties ​ values string[] The host value.

Copy

curl --location 'https://api.novita.ai/v3/assets/training_dataset' \
--header 'Authorization: Bearer {{API Key}}' \
--header 'Content-Type: application/json' \
--data '{
"file_extension": "png"
}'
