Convert Pay-As-You-Go Instance to Subscription cURL Copy
```
curl --request POST \
--url https://api.novita.ai/gpu-instance/openapi/v1/gpu/instance/transToMonthlyInstance \
--header 'Authorization: ' \
--header 'Content-Type: ' \
--data '
{
"instanceId": " ",
"month": 123
}
'
```

## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ instanceId string required The ID of the pay-as-you-go instance to be converted to a subscription instance.
