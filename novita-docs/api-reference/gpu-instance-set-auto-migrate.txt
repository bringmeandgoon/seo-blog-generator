Set Auto Migrate Strategy cURL Copy
```
curl --request POST \
--url https://api.novita.ai/gpu-instance/openapi/v1/gpu/instance/setAutoMigrate \
--header 'Authorization: ' \
--header 'Content-Type: ' \
--data '
{
"instanceId": " ",
"autoMigrateOpen": true,
"autoMigrateSystemDisk": true
}
'
```

## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ instanceId string required Instance ID.
​ autoMigrateOpen boolean required Enable/disable automatic instance migration.
