## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ template object required Template settings. Hide properties ​ name string required Template name. String, length limit: 1-255 characters. ​ readme string Template README content (in Markdown format). String, length limit: 0-102400 characters. ​ type string required Template type.
Enum: instance ​ channel string required Template channel.
Enum: private ​ image string required Docker image address for instance startup. String, length limit: 1-500 characters. ​ imageAuth string Image repository authentication ID for private images. String, length limit: 0-255 characters. ​ startCommand string Startup command for the instance. String, length limit: 0-2047 characters. ​ entrypoint string Instance startup entrypoint. This setting will override the Docker image’s ENTRYPOINT. String, length limit: 0-2047 characters. ​ rootfsSize integer required Root filesystem storage (GB). Integer, value must be greater than 0. ​ ports object[] Exposed port settings. Hide properties ​ type string required Exposed port types.
Enum: http , tcp ​ ports integer[] required Exposed ports (maximum of 10). Supported port range: 1-65535, except for 2222, 2223, 2224 which are reserved for internal use. ​ envs object[] Environment variables injected into the instance. Up to 100 environment variable pairs can be created. Hide properties ​ key string Environment variable key. String, length limit: 0-511 characters. ​ value string Environment variable value. String, length limit: 0-4095 characters. ​ minCudaVersion string Minimum supported CUDA version. String, length limit: 0-255 characters. Format example: 11.8, 12.4.
## ​Response
​ templateId string ID of the created template.
