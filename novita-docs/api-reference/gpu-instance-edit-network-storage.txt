## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ storageId string required The unique identifier for the storage.
​ storageName string required The name of the storage.
​ storageSize string required The size of the storage in appropriate units.
