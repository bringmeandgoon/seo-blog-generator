## ​Description
After deleting an endpoint, all worker processes associated with this endpoint will be automatically removed. Please proceed with caution.
## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
