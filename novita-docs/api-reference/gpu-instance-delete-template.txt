## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ templateId string required The ID of the template to be deleted.
## ​Response
​ templateId string The ID of the deleted template.
