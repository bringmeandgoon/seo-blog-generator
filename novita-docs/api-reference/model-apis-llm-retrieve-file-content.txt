Retrieves the content of a specific file using its file ID. This is commonly used to download batch output files or error files.
## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Path Parameters
​ file_id string required The unique identifier of the file to retrieve content for.
## ​Response
The response returns the raw file content. For batch output files, this will be in JSONL format where each line contains a batch request result.
Example Response
