## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Response
​ total integer required Number of created prewarm tasks.
​ limit integer required Maximum number of prewarm tasks that can be created.
