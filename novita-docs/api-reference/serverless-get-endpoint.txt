Hide properties ‚Äã minNum integer required Minimum number of workers. ‚Äã maxNum integer required Maximum number of workers. ‚Äã freeTimeout string required Idle timeout, in seconds. ‚Äã maxConcurrent string required Maximum concurrency. ‚Äã gpuNum integer required Number of GPUs per worker. ‚Äã cudaVersion string required CUDA version. ‚Äã policy object required Auto-scaling policy for the endpoint. Hide properties ‚Äã type string required Policy type. Options:
- queue: Queue latency policy, adjusts the number of workers based on the waiting time of requests in the queue.
- concurrency: Queue request policy, automatically adjusts the number of workers based on the number of requests in the queue.
‚Äã value string required The meaning of value depends on the type:
- If type = queue, value is the queue waiting time in seconds.
- If type = concurrency, value is the maximum number of requests in the queue.
‚Äã image object required Image configuration for the endpoint. - local: Local storage.
