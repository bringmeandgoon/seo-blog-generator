## ‚ÄãRequest Headers
‚Äã Content-Type string required Enum: application/json
‚Äã Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ‚ÄãQuery Parameters
‚Äã cycleType string required Billing cycle granularity. Options: Hour , Day , Week , Month
‚Äã productCategory string required Product type. Options:
- summary (Summary bill)
- gpu (GPU instance)
- llm (llm api)
- serverless (Serverless Endpoint)
- cloud_storage (Storage resources)
- gen_api (Image/Video/Audio generation)
‚Äã productName string Product name. Supports fuzzy matching.
‚Äã startTime string Start time of the billing period to query, timestamp in seconds, default: 0.
‚Äã endTime string End time of the billing period to query, timestamp in seconds, default: 0.
‚Äã ownerId string Specify instance ID to query.
## ‚ÄãResponse Parameters
‚Äã bills object[] required Pay-as-you-go instance billing information. Hide properties ‚Äã userId string required User ID to which the instance belongs. ‚Äã startTime string required Start time of the bill. Format is Unix timestamp. ‚Äã endTime string required End time of the bill. Format is Unix timestamp. ‚Äã billingMethod string required Billing method of the instance. A value of 1 indicates pay-as-you-go. ‚Äã productName string required Product name. ‚Äã category string required Product category. ‚Äã ownerID string required Instance ID. ‚Äã billNum0 string required
- llm: input token
- gpu: number of cards * billing time
- storage: storage size * billing time
‚Äã billNum1 string required
- llm: output token
‚Äã basePrice0 string required Original Price ‚Äã basePrice1 string required No significance ‚Äã discountPrice0 string required
- llm: input token unit price
- others: unit price
‚Äã discountPrice1 string required
- llm: output token unit price
‚Äã amount string required Total price. ‚Äã voucherAmount string required Voucher deduction amount. ‚Äã payAmount string required Cash payment amount. ‚Äã payAmountDisplay string required ‚Äã pricePrecision string required Price Precision. ‚Äã productId string required Product ID.
