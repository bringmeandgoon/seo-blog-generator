Deletes a specific file using its file ID.
## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Path Parameters
​ file_id string required The unique identifier of the file to delete.
## ​Response
​ deleted boolean required Indicates whether the file was successfully deleted. Returns true if the deletion was successful.
​ id string required The unique identifier of the deleted file.
