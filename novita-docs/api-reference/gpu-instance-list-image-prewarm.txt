## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Query Parameters
​ page integer Page number. Default: 1.
​ pageSize integer Page size. Default: 10.
​ state string Prewarm task state. Valid values: Pending , Running , Succeeded , Failed .
​ clusterId string Cluster ID.
​ name string Image name or task note for fuzzy matching.
## ​Response
​ data object[] required List of prewarm tasks. Hide properties ​ id string required Prewarm task ID. ​ imageName string required Image name (the last segment of the image URL without the tag). ​ imageUrl string required Image URL. ​ repositoryAuth string Image registry authentication ID. ​ clusterId string required Cluster ID. ​ clusterName string required Cluster name. ​ products object[] required Products to prewarm. Show properties ​ productId string required Product ID. ​ productName string required Product name. ​ imageSize string Image size in bytes. ​ createTime string required Task creation time (Unix timestamp in seconds). ​ state string required Task state. Valid values: Pending , Running , Succeeded , Failed . ​ completeTime string Task completion time (Unix timestamp in seconds). ​ note string Task note. ​ reason string[] Reason messages.
