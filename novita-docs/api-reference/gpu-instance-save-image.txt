## ​Request Headers
​ Content-Type string required Enum: application/json
​ Authorization string required Bearer authentication format, for example: Bearer {{API Key}}.
## ​Request Body
​ instanceId string required Instance ID. String, length: 1-255 characters.
​ image string required Target image address, including registry/repository:tag. String, length: 1-4095 characters.
​ registryAuthId string Container registry authentication ID. Not required for public registries or platform-provided registries. Required for third-party private registries.
